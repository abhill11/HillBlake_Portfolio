Python

GIS tool created using python scripting.

An automated tool used to batch-clip layers to a specified boundary. The tool projects all layers in a folder to the same projected coordinate system before clipping them to the extent of another specified layer and saving them to a new geodatabase. The tool will save the intermitent, projected layers in a separate folder than the original or final outputs. The parameters provide helpful guidance and error feedback. The tool and script are available for download from my github. You may download the zip file or individual components.
